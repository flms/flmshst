Does developing softwares for touch screen devices differ from other devices?

Yes, touch screen softwares use another technology to develop UI and it's different with ordinary desktop softwares. for example tablet applications have designed for touch screens

Is there a difference between running and using a typical software vs touch software?

No, you can use both software like eachother but for typical softwares we use mouse but for softwares which is designed for touch screens we can use our hands and it would improve our performance.

Should we also buy hardwares from faralams?

No, you can have your own hardware if you want to.
