Point of Sale

A productive and beautiful designed software to be used on touch-screen devices

Unlike current POS softwares, Faralams POS designed from base to be used on touch devices. By using cutting edge technology to develop this software we've been able to use screen space better.

  • Beautiful Design

    By analyzing work flow of Restaurant POS users and by using an special design team, we designed this software so you would enjoy using an amazing and interesting software.

  • High Performance

    Easy to learn and fast to register and get orders from digital menus at the least possible time, made this software the best choice for restaurants.

  • Touch Ready

    With the fact that the Faralams products are designed to be used by touch-screen devises, it fairly will increase your speed and productivity multiple times by using it.

  • Customer Information

    After registering orders, you can enter customer's information and receipt's details

  • Quick Access

    To filter tables by receipt code or table number you can use this field

  • Foods categories

    You can easily move between different categories without any restriction on category numbers

  • Quick Select

    With just one click you can select your order

  • Merging Orders

    In case of needing to merge few orders and print them in one receipt, you can Merge them easily.

  • Touch Numpad Key

    If you need to have more than a few orders for an item, you can enter any number without just clicking on items

  • New and Open receipts list

    You can see a list of New or Open orders on this section and send them to kitchen printer or close that table

  • Customer receipt

    After opening a receipt, you can see receipt details and if it doesn't need to change, print it
